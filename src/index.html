BLUR — Децентрализованная программа лояльности

ERC20 токен, реализующий программу лояльности на основе технологии блокчейн. Он объединяет несколько ТСП в единую сеть, этим универсальным токеном можно пользоваться в каждой из участвующих ТСП. Токены начисляются и могут быть потрачены только при покупке по безналичному рассчету.
